=== Dependencies ===
 * Python 2.6

 * SimpleParse 2.1
   http://simpleparse.sourceforge.net/

 * omniORB 4.1.4 (optional)
   http://omniORB.sourceforge.net

=== INSTALLATION ===

    python setup.py install --prefix $YOUR_PREFIX

=== USAGE ===
 * Stand-alone mode
 * Server-client mode (requires omniORB (corbaserver))
